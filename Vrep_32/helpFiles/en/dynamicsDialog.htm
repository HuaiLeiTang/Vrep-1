Dynamics dialog

The dynamics dialog is part of the calculation module properties dialog, which is located at [Menu bar --> Tools --> Calculation module properties]. You can also open the dialog with a click on its toolbar button:

[Calculation module properties toolbar button]


In the calculation module properties dialog, click the Dynamics button to display the dynamics dialog:

[Dynamics dialog]


  • Dynamics enabled: allows enabling/disabling dynamics calculations for the entire scene.
  • Engine to use: the dynamics engine to use. Bullet stands for the Bullet Physics Library and ODE stands for the Open Dynamics Engine. Each engine has specific strengths and weaknesses. A simulation running fine with one engine doesn't always mean that it will also run fine with the other engine. Each engine has specific parameters that can be set globally (see hereafter) or locally (in the shape dynamics dialog or in the joint dynamics dialog). Make sure you read the respective engine documentations to achieve best parameter settings.
  • Display contact points: allows displaying contact points between respondable shapes.
  • Adjust engine parameters: opens the dynamics engine dialog, which allows adjusting engine-specific parameters.
  • Gravity: the amplitude and direction of a constant force applied to all non-static shapes (e.g. gravity).
